How to build lldbPluginABIMSP430 library

2023-10-16 LLDB-Plugin MSP430 

The ABI libraries in build dir

    ABI 
cd build
cat ./tools/lldb/source/Plugins/Plugins.def | grep "LLDB_PLUGIN("  | grep ABI

LLDB_PLUGIN(ABIARM)
LLDB_PLUGIN(ABIPowerPC)
LLDB_PLUGIN(ABIX86)
LLDB_PLUGIN(CXXItaniumABI)

The env LLVM_TARGETS_TO_BUILD controls whether the target library be built

  • llvm\lib\TargetCMakeLists.txt

    • foreach(t ${LLVM_TARGETS_TO_BUILD})
message(STATUS "Targeting ${t}")
add_subdirectory(${t})
endforeach()
