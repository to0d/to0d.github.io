        5.6. MC 层 LLVM-MC

        ﻿
      • 5.6.1. How disassembly works
        •
        • LLVM–MC (machine code) 层位于LLVM的底层，主要功能是负责汇编 (assembly)、反汇编 (disassembly)、以及生成二进制文件  LLVM-MC

        • 按照输入不同主要分为两条路径
          • 输入为汇编文件，经过Assembly Parser将汇编文件中的指令解析成Operand，然后通过MCTargetAsmParser将Operand转换成对应的MCInst,最终进入到MCStreamer,通过Instruction Encoder生成二进制文件。
          • 输入为二进制文件，经过Instruction Decoder，将二进制代码反汇编生成MCInst, 最终进入到MCStreamer按照需要生成汇编文件。  disassembly

        5.6.1. How disassembly works

        • LLVM MC: 输入为二进制文件，经过Instruction Decoder，将二进制代码反汇编生成MCInst, 最终进入到MCStreamer按照需要生成汇编文件
        • GUN: objdump gdb 也是使用它进行反汇编
