    • ELF–Section (name=节, ptag=ELF–Struct,
      • ELF 文件 中可以处理的最小不可分割单位
      • 包含链接视图的批量目标文件信息。此数据包括指令、数据、符号表和重定位信息
      • Linking View
      )        ELF-Section ELF-Struct

    • ELF–Segment (name=段, ptag=ELF–Struct, ELF-Section节的集合。由 exec 或 linker 映射到内存映像的最小独立单元, Execution View )  ELF-Segment ELF-Struct

    • ELF–PHT (name=Program Header Table, ptag=ELF–Struct, 程序头表 : 指示系统如何创建进程映像。 用于生成进程映像、可执行文件和共享目标文件的文件必须具有 程序头表 。 可重定位目标文件无需程序头表。)  ELF-PHT ELF-Struct

    • ELF–SHT (name=Section Header Table, ptag=ELF–Struct, 节头表* : 包含说明文件各节的信息。 每节在表中有一个与之对应的项。每一项都指定了节名称和节大小之类的信息。 链接编辑过程中使用的文件必须具有节头表。 )  ELF-SHT ELF-Struct
