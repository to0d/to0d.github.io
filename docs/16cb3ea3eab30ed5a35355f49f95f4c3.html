Install Git on Linux

Git Install 

  1. yum install git

    2. git --version    #1.7.1-3.el6_4.1	
git config --global usr.name "liuyan"
git config --global usr.email "liuyan3000@hotmail.com"
  2. 在LINUX上创建GIT服务器

    • 生成 SSH 公钥:

      • ssh-keygen -t rsa ==> id_dsa.pub
    • adduser git

      • passwd git
su git
cd ~
mkdir .ssh
    • 把开发者的 SSH 公钥添加到这个用户的 authorized_keys 文件中, 免密码

      • cat ~/.ssh/id_rsa.pub >> ~git/.ssh/authorized_keys
chown -R git /git/work

vi /etc/ssh/sshd_config
RSAAuthentication yes            #开启RSA认证功能
PubkeyAuthentication yes      #开启公匙认证
StrictModes no

service sshd restart

su git
cd /git
mkdir work.git
cd work.git
git --bare init
  3. 禁用shell登录

    4. vi /etc/passwd ==> git-shell
git:x:1001:1001:,,,:/home/git:/usr/bin/git-shell
  4. add windows ssh–key to server

    5. > Git-bash
> ssh-keygen -t rsa -C "liuyan3000@hotmail"
  5. git url format

    6. git@9.123.135.91:/git/work.git
