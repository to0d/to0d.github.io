Git可以在你提交时自动地把行结束符 CRLF 转换成 LF , 而在签出代码时把LF转换成 CRLF . 用 core.autocrlf 来打开此项功能, 如果是在Windows系统上, 把它设置成 true , 这样当签出代码时, LF会被转换成 CRLF 

    $ git config --global core.autocrlf true

修改git设置 core.autocrlf=input 检出时不转换, 提交转换为 LF, 这样可以避免提交windows换行符的情况 

    $ git config --global core.autocrlf input
