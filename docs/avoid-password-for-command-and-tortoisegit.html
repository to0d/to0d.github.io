How to avoid inputing password for command line & TortoiseGit

2020-01-12 Git TortoiseGit ssh 

TortoiseGit setting

  • Create public key (git.key.txt) & private key (git.key.ppk) via puttygen

  • Copy them to folder C:\Users\<username>\.ssh

  • Add public key to server
    • For linux server: ˜/.ssh/authorized_key
    • For GitHub: Gitub – Your profile – SSH and GPG keys

Command line setting

  • Load previous private key (git.key.ppk) via puttygen

  • Conversions –> Export OpenSSH Key, save to C:\Users\<username>\.ssh\id_rsa

  • Re–generate OpenSSH public key via ssh–keygen  OpenSSH

    • ssh-keygen -f .ssh/id_rsa -y .ssh/id_rsa.pub
