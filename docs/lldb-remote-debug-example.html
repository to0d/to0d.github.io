lldb remote debug example

2023-10-17 LLDB Remote-Debug 

  • start gdb server ( without program)  gdb-server

    • > gdbserver --multi 127.0.0.1:8059
Listening on port 8059
  • start lldb

    • > lldb
  • select platform  lldb-platform-select

    • (lldb) platform list
Available platforms:
host: Local Linux user platform plug-in.
...
remote-gdb-server: A platform that uses the GDB remote protocol as the communication transport.
(lldb) platform select remote-gdb-server
Platform: remote-gdb-server
Connected: no
  • connect remote target

    • (lldb) platform connect connect://127.0.0.1:8059
Platform: remote-gdb-server
Hostname: (null)
Connected: yes
  • debugging

    • (lldb) file a.out
Current executable set to '/home/todd-wsl/a.out' (x86_64).
(lldb) l
4    {
5        int x = 0;
6        x += 2;
7        printf("hello, x=%d\n", x);
8    }
(lldb) b 7
Breakpoint 1: where = a.out`main + 23 at test.c:7:5, address = 0x0000000000001160
(lldb) r
Process 443 launched: '/home/todd-wsl/a.out' (x86_64)

    • log list
